# Homebrew Tap for stem-splitter

This repository contains the Homebrew formula for **stem-splitter** â€”
an AI-powered audio stem separation CLI.

## Installation

First, add the tap:

```bash
brew tap gentij/stem-splitter
```

Then install the CLI:

```bash
brew install stem-splitter
```

## Usage

After installation, the `stem-splitter` command will be available in your shell:

```bash
stem-splitter --version
stem-splitter list
```
